TFRecord and tf.train.Example¶

In this tutorial, I will first give a brief introduction of TFRecord and go over how to create, save and read TFRecord data, a simple format for storing a sequence of binary records.

Here are the advantages of using this data format:

  1. More efficient storage: the TFRecord data can take up less space than the original data; it can also be partitioned into multiple files.
  2. Fast I/O: the TFRecord format can be read with parallel I/O operations, which is useful for TPUs or multiple hosts. TPUs require data to be stored remotely (e.g. on Google Cloud Storage) and using the TFRecord format makes it easier to load the data without batch-downloading.
  3. Self-contained files: the TFRecord data can be read from a single sourceâ€”for example, the COCO2017 dataset originally stores data in two folders ("images" and "annotations").
  4. Better Performance with TF APIs: using tf.data API will get better performance.
  5. RAM Friendly: only the data required at the time (e.g. a batch) is loaded from disk for processing

A TFRecord dataset contains a sequence of tf.train.Example messages. The tf.train.Example message is a {"string": tf.train.Feature} mapping. tf.train.Feature can accept the following three types: tf.train.BytesList(a list of string/byte), tf.train.FloatList(a list of float/double), tf.train.Int64List(a list of boolen/int/uint)

We will create a fake dataset with the following properties for playing with TFRecord data format:

  1. image id: int
  2. categories: list
  3. published year: int
  4. description: string
  5. value: float
  6. image: jpg

def int64_feature(value): 
    return tf.train.Feature(
        int64_list=tf.train.Int64List(value=[value])
    )


def string_feature(value):
    return tf.train.Feature(
        bytes_list=tf.train.BytesList(value=[value.encode()])
    )


def int64_feature_list(value):
    return tf.train.Feature(
        int64_list=tf.train.Int64List(value=value)
    )


def float_feature(value):
    return tf.train.Feature(
        float_list=tf.train.FloatList(value=[value])
    )


def image_feature(value):
    return tf.train.Feature(
        bytes_list=tf.train.BytesList(value=[tf.io.encode_jpeg(value).numpy()])
    )


def create_example(image, path, example):
    feature = {
        "image_id": int64_feature(example['image_id']),
        "categories": int64_feature_list(example['categories']),
        "published_year": int64_feature(example['published_year']),
        "description": string_feature(example["description"]),
        "value": float_feature(example["value"]),
        "image_path": string_feature(path),
        "image": image_feature(image)
    }
    return tf.train.Example(features=tf.train.Features(feature=feature))

def parse_tfrecord_fn(example):
    # tf.io.FixedLenFeature(shape, dtype, default_value=None): Configuration for parsing a fixed-length input feature.
    # tf.io.VarLenFeature(dtype): Configuration for parsing a variable-length input feature.
    feature_description = {
        "image_id": tf.io.FixedLenFeature([], tf.int64),
        "categories": tf.io.VarLenFeature(tf.int64),
        "published_year": tf.io.FixedLenFeature([], tf.int64),
        "description": tf.io.FixedLenFeature([], tf.string),
        "value": tf.io.FixedLenFeature([], tf.float32),
        "image_path": tf.io.FixedLenFeature([], tf.string),
        "image": tf.io.FixedLenFeature([], tf.string),
    }
    example = tf.io.parse_single_example(example, feature_description)
    example["image"] = tf.io.decode_jpeg(example["image"], channels=3)
    return example

Generate data in the TFRecord format¶

In [6]:
images_dir = 'images'

for tfrec_num in range(num_tfrecods):
    samples = annotations[(tfrec_num * num_samples) : ((tfrec_num + 1) * num_samples)]
    
    with tf.io.TFRecordWriter(
        tfrecords_dir + "/file_%.2i-%i.tfrec" % (tfrec_num, len(samples))
    ) as writer:
        for sample in samples:
            image_path = f"{images_dir}/{sample['image_id']}.jpeg"
            image = tf.io.decode_jpeg(tf.io.read_file(image_path))
            example = create_example(image, image_path, sample)
            writer.write(example.SerializeToString())
